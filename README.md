# Fundamentos de Programación - Java

Este repositorio contiene 10 ejercicios resueltos, realizados como práctica del libro **Fundamentos de Programación: Diagramas de flujo, Diagramas N-S, Pseudocódigo y Java**.

## Descripción

El objetivo de este trabajo es aplicar conceptos básicos de programación mediante ejercicios desarrollados en Java.  
Los programas utilizan entrada de datos por teclado, variables, operadores, condicionales, ciclos y estructuras básicas de control.

## Ejercicios incluidos

1. **Área de un círculo**  
   Calcula el área de un círculo a partir del radio ingresado por el usuario.

2. **Conversión de temperatura**  
   Convierte una temperatura de grados Celsius a Fahrenheit.

3. **Número par o impar**  
   Determina si un número entero es par o impar.

4. **Mayor de tres números**  
   Solicita tres números y muestra cuál es el mayor.

5. **Factorial de un número**  
   Calcula el factorial de un número entero positivo.

6. **Tabla de multiplicar**  
   Muestra la tabla de multiplicar de un número del 1 al 10.

7. **Suma de números del 1 al N**  
   Calcula la suma de todos los números desde 1 hasta un número N.

8. **Número primo**  
   Verifica si un número ingresado es primo o no.

9. **Promedio de calificaciones**  
   Calcula el promedio de varias calificaciones e indica si el estudiante aprueba o reprueba.

10. **Serie Fibonacci**  
    Muestra los primeros N términos de la serie Fibonacci.
